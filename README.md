## Urls

You can access the services in following urls. Check `docker-compose.yml` for available containers.

-   `localhost` (Laravel app)
-   `localhost:1025` (MailHog)
-   `localhost:8080` (Phpmyadmin)
-   `localhost:5173` (Vite server)
